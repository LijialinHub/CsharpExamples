# C# 学习项目目录

## 项目大纲

### 最近项目

#### 2025-12-12 - 拓展方法和LINQ基本用法
学习拓展方法和LINQ基本用法。

#### 2025-12-11 - 委托和线程
学习委托(Delegate)的基本概念、多播功能、泛型委托、匿名方法和Lambda表达式，以及系统提供的Action和Func委托。

#### 2025-12-10 - 图形变换

#### 2025-12-08 - 窗体内部塞窗体

#### 2025-12-05 - 使用Ini文件保存参数
实现INI配置文件读写功能，用于保存和读取应用程序参数设置。

#### 2025-12-04 - ListView控件应用
使用ListView控件展示文件和文件夹信息，包括递归获取目录内容。

#### 2025-12-03 - 文件文件夹路径操作
文件和文件夹的基本操作，包括创建、复制、移动、删除和读写功能。

#### 2025-12-01 - Xml序列化
实现对象的XML序列化和反序列化，用于数据持久化存储。
