# C# 和 WinForm 学习示例项目

这是一个用于学习C#语言和WinForm桌面应用程序开发的示例集合，按日期组织代码练习，涵盖从基础语法到UI设计、面向对象编程、文件操作等多个方面。

## 项目结构说明

以下是按时间顺序排列的各目录内容概述：

### 2025-11-08 - C#基础语法入门
介绍了命名空间、类、字段、属性等基本概念，以及C#中的注释方法。

### 2025-11-10 - 构造函数、方法和static关键字
讲解了构造函数（无参和有参）、方法定义与调用、static关键字的使用，以及简单数据类型和逻辑运算。

### 2025-11-11 - 数据类型深入和枚举
涵盖了整数类型按位运算、进制转换、数据类型转换方法（Parse、TryParse）、枚举类型的定义和使用。

### 2025-11-12 - 常量、装箱拆箱和控制台输入
介绍了const常量、装箱拆箱概念、字符类型char、控制台输入Console.ReadLine()及相关练习。

### 2025-11-13 - 运算符和控制结构
讲解了异或运算、switch-case语句、while和do-while循环、break语句，以及一维数组的使用。

### 2025-11-14 - 程序控制和Math类
介绍了break、continue、return、goto等控制关键字，Math类的常用方法，以及二维数组的使用。

### 2025-11-15 - 字符串和转义字符
讲解了九九乘法表的实现、三维数组、转义字符及其使用方法。

### 2025-11-17 - 字符串操作和参数传递
深入介绍了字符串的各种操作方法、StringBuilder与String的区别、方法参数传递机制（值类型和引用类型）。

### 2025-11-18 - 三元运算符和泛型集合
讲解了三元运算符、List集合的使用、泛型概念、随机数Random类的应用。

### 2025-11-19 - 异常处理和DLL引用
介绍了try-catch异常处理机制，以及如何创建和引用DLL程序集。

### 2025-11-19(dll) - 计算类库
一个独立的计算类库项目，提供了泛型加法和减法运算方法。

### 2025-11-20 - 类的继承和多态
讲解了类的继承、里氏转换原则、is和as关键字、虚方法和重写方法、结构体的概念。

### 2025-11-21 - 抽象类和正则表达式
介绍了抽象类和抽象方法、工厂模式的实现，以及正则表达式的使用方法。

### 2025-11-24 - 正则表达式练习
通过实际案例练习正则表达式的使用，如日期格式验证和G代码文件中坐标提取。

### 2025-11-24(2) - G代码解析器
使用WinForm创建了一个简单的G代码文件读取和坐标解析应用程序。

### 2025-11-25 - WinForm基础控件
演示了WinForm中常用控件的使用，包括按钮事件、对话框、MDI多文档界面等。

### 2025-11-27 - 文件操作和表单验证
展示了文件操作（File类）、表单数据收集和验证、错误提示等实用功能。

### 2025-11-28 - 图像处理
实现了图片浏览功能，包括打开、切换、保存图像等操作。

### 2025-12-01 - 员工管理系统
完整的员工管理系统，包含添加、修改、删除员工信息，使用XML文件进行数据持久化。

### 2025-12-03 - 文件和目录操作
讲解了File和Directory类的使用，包括文件和目录的创建、复制、移动、删除等操作。

### 2025-12-04 - 文件浏览器
使用ListView控件实现了一个简单的文件浏览器，能够递归显示文件夹中的文件信息。

### 2025-12-05 - INI配置文件操作
提供了INI配置文件读写的辅助类，通过Windows API实现配置项的读写功能。

### 2025-12-08 - 多页面应用程序
展示如何创建多页面的WinForm应用程序，通过面板切换不同的功能页面。

### 2025-12-09 - GDI+绘图基础
介绍了使用GDI+进行基本图形绘制的方法，包括线条、矩形、圆形、多边形等。

### 2025-12-10 - 坐标变换和高级绘图
深入讲解了坐标变换（平移、旋转、缩放、镜像）在绘图中的应用。

### 2025-12-11 - 委托和Lambda表达式
详细介绍了委托的概念、多播委托、泛型委托、匿名方法和Lambda表达式，以及系统提供的Action和Func委托。
